# Resumen Rapido — Estudiante / Acudiente

One-pager con lo esencial del rol.

| Campo | Valor |
|---|---|
| Rol | Estudiante / Acudiente (ROL-09) |
| Objetivo | Permitir consulta autonoma de datos academicos al estudiante y al acudiente |
| Permisos Clave | Solo lectura. Acceso al portal: configurable (puede estar desactivado por colegio) |
| Dispositivo | Mobile principalmente; desktop para descargas |
| Acciones Principales | Consultar notas; consultar horario; recibir comunicados; consultar boletines; consultar observaciones (configurable) |
| Nivel | Minimo (solo consulta) |
