# Resumen Rapido — Coordinador Academico y Convivencia (combinado)

One-pager con lo esencial del rol.

| Campo | Valor |
|---|---|
| Rol | Coordinador Academico y Convivencia (combinado) (ROL-05) |
| Objetivo | Cubrir academico + convivencia desde un solo rol en colegios pequenos |
| Permisos Clave | Union de ROL-03 + ROL-04. Mutuamente excluyente con esos roles separados (RR-12) |
| Dispositivo | Desktop principalmente |
| Acciones Principales | Plan de estudios + grupos + horarios + consolidados + cierre + observador + tipologias + reportes de convivencia |
| Nivel | Alto (equivalente a ROL-03 + ROL-04) |
