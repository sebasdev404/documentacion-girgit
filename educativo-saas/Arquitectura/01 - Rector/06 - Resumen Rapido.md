# Resumen Rapido — Rector / Administrador del Colegio

One-pager con lo esencial del rol.

| Campo | Valor |
|---|---|
| Rol | Rector / Administrador del Colegio (ROL-02) |
| Objetivo | Configurar el colegio y administrarlo en el dia a dia |
| Permisos Clave | Acceso total dentro del tenant. CRUD sobre configuracion, usuarios, roles. Lectura completa sobre operacion |
| Dispositivo | Desktop principalmente; mobile para aprobaciones |
| Acciones Principales | Configurar identidad, calendario, jornadas, escala valorativa, metodo de aprobacion; crear usuarios; asignar roles; ver reportes y logs |
| Nivel | Maximo dentro del tenant |
