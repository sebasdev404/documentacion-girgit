# Resumen Rapido — Secretaria Academica

One-pager con lo esencial del rol.

| Campo | Valor |
|---|---|
| Rol | Secretaria Academica (ROL-06) |
| Objetivo | Gestionar matricula y emitir documentos oficiales del colegio |
| Permisos Clave | CRUD sobre estudiantes y acudientes. Reportar (generar) constancias, certificados, paz y salvos, SIMAT |
| Dispositivo | Desktop principalmente (impresion, formularios extensos) |
| Acciones Principales | Inscribir estudiantes; cargar documentos; asignar a grupo; generar constancias / certificados / paz y salvos; reportar SIMAT |
| Nivel | Alto en operacion administrativa |
