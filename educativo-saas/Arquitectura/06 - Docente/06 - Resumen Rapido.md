# Resumen Rapido — Docente

One-pager con lo esencial del rol.

| Campo | Valor |
|---|---|
| Rol | Docente (ROL-07) |
| Objetivo | Operar el aula: notas, asistencia y observaciones de sus materias asignadas |
| Permisos Clave | Editar notas y asistencia SOLO en materias y grupos asignados. Sin acceso a otros grupos (RR-10) |
| Dispositivo | Mobile + desktop. Mobile comun para asistencia y notas |
| Acciones Principales | Registrar/editar notas; registrar asistencia; agregar observaciones academicas; consultar horario propio; ver listado de estudiantes asignados |
| Nivel | Limitado a sus asignaciones |
