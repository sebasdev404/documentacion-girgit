# Resumen Rapido — Coordinador Academico

One-pager con lo esencial del rol.

| Campo | Valor |
|---|---|
| Rol | Coordinador Academico (ROL-03) |
| Objetivo | Mantener la estructura academica operativa del colegio organizada |
| Permisos Clave | CRUD sobre plan de estudios, grupos, horarios. Lectura de consolidados. Edicion de notas: configurable |
| Dispositivo | Desktop principalmente |
| Acciones Principales | Gestionar plan de estudios; crear grupos; asignar docentes; construir horarios; ver consolidados; gestionar nivelaciones; coordinar cierre de periodo |
| Nivel | Alto en operacion academica |
