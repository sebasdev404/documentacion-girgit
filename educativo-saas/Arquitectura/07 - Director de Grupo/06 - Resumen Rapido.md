# Resumen Rapido — Director de Grupo (complemento)

One-pager con lo esencial del rol.

| Campo | Valor |
|---|---|
| Rol | Director de Grupo (complemento) (ROL-08) |
| Objetivo | Acompanar el grupo dirigido (complemento al rol Docente) |
| Permisos Clave | Permisos ADICIONALES sobre el grupo dirigido. En otros grupos sigue siendo solo Docente (RR-09) |
| Dispositivo | Mobile + desktop |
| Acciones Principales | Ver consolidado completo del grupo; generar boletines; agregar observacion general en boletin; registrar anotaciones en el observador del grupo |
| Nivel | Combina Docente + acceso ampliado al grupo dirigido |
